# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto segue [Versionamento Semântico](https://semver.org/pt-BR/).

## [1.0.0] - 2026-06-14

### ✨ Adicionado
- Aplicação **Orenji Focus** - Gerenciamento de foco e produtividade com técnicas de pomodoro
- Aplicação **Orenji Habit** - Rastreador de hábitos para construir rotinas positivas
- Módulo **Orenji Shared** - Componentes e utilitários compartilhados entre aplicações
- Logos e assets visuais para ambas as aplicações
- README.md com documentação do projeto
- Interfaces de dashboard, índice e configurações para ambas as apps
- Sistema de timer com suporte a múltiplos métodos de foco
- Estilos responsivos com CSS customizado
- Scripts JavaScript para funcionalidades interativas

### 🎨 Melhorado
- Estrutura de headers com branding consistente
- Implementação de UI para seleção de métodos de foco (ex: Pomodoro 25/5)
- Estilos responsivos para componentes principais (header, nav, brand)
- Paths atualizados para componentes compartilhados

### 🔧 Corrigido
- Integração de caminhos para estilos e scripts compartilhados
- Comportamento responsivo de elementos header
- Sincronização de funcionalidades entre versão pública e desenvolvimento
